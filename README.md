# okashi

市販お菓子の栄養成分・価格データを置いているリポジトリです。

Colab等から `pd.read_excel(URL)` で直接読み込めるように、Excelファイルを置いています。

---

## 収録データ

### `okashi.xlsx`

| 項目 | 内容 |
|---|---|
| 対象商品 | 国内で市販されている32種類のお菓子（チョコ系・スナック系・キャンディ系・米菓系 各8品） |
| 含まれる列 | 商品名、ジャンル、値段（円）、カロリー（kcal）、たんぱく質（g）、脂質（g）、炭水化物（g）、内容量（g）、塩分（g）、添加物、ソース（URL） |
| 行数 | 32行 |

### 使い方の例

```python
import pandas as pd
url = 'https://raw.githubusercontent.com/ctxzz/okashi/main/okashi.xlsx'
df = pd.read_excel(url)
df.head()
```

---

## 出典

すべての数値は、各メーカーの**公式Webサイト**で公開されている情報を取得・記録したものです。各商品の出典URLはデータ内の **「ソース（URL）」列** に1件ごとに記載しています。

## 取得時期

2026年5月

## データの正確性について

商品リニューアル等により、現時点の実際の値と異なる場合があります。正確な栄養成分・価格については、各メーカーの公式Webサイトでご確認ください。

## 商標・知的財産について

データ内に記載されている商品名・ブランド名（ガーナ、ポッキー、亀田の柿の種、ハイチュウ など）は、それぞれ各社の登録商標または商標です。本データの公開は、食品表示法等で各メーカーが公開している事実情報を、教育目的で参照可能な形に整理したものです。**各メーカーから公認・推奨を受けたものではありません。**
